## 설치

```shell
sudo apt-get update
```

```shell
sudo apt-get install certbot
```

```shell
sudo apt-get install python3-certbot-nginx
```

## Nginx

```shell
sudo vim /etc/nginx/conf.d/default.conf
```

```nginx
server {
    listen 80;
    listen [::]:80;
    return 301 https://$host$request_uri;
}
```

## certbot

```shell
sudo certbot --nginx \
    -d wiki.loliot.net \
    -d loliot.net
```

자동으로 `/etc/letsencrypt`에 인증관련 파일을 생성하고, `/etc/nginx/conf.d/default.conf` 내용도 수정을 해줍니다.

## Renew

```shell
sudo certbot renew
```
