## 비공계 키 생성

`프로젝트 설정` -> `서비스 계정` -> `Firebase Admin SDK` -> `새 비공개 키 생성`

## 커스텀 이메일

이메일 인증, 비밀번호 재설정 등 이메일을 활용할 때, Firebase에서 기본으로 제공하는 이메일 템플릿과 이메일 응답에 따른 처리 페이지는 매우 간단합니다. 사용자 경험을 개선하기 위해 이를 커스텀할 필요가 있지만 이는 Firebase에서 제공하는 기능이 아니기 때문에 직접 구현해야 합니다.

### 이메일 보내기
