## Github Package

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: github-image-pull-secret
  namespace: <namespace>
type: kubernetes.io/dockerconfigjson
stringData:
  .dockerconfigjson: "{"auths":{"ghcr.io":{"auth":"<base64 username:token>"}}}"
```

## Usage

```yaml
spec:
  containers: ...
  imagePullSecrets:
    - name: <secret-name>
```
