## 설치

```shell
helm repo add strimzi https://strimzi.io/charts/
```

```shell
helm repo update strimzi \
&& helm search repo strimzi/strimzi-kafka-operator -l | head -n 10
```

```shell
helm show values strimzi/strimzi-kafka-operator \
    --version 0.32.0 \
    > kafka-operator-values.yaml
```

```yaml title="kafka-operator-values.yaml"
labels: {}

resources:
  limits:
    memory: 384Mi
  requests:
    memory: 384Mi
    cpu: 200m

extraEnvs:
  - name: JAVA_OPTS
    value: "-Xms256m -Xmx256m"
```

```shell
helm template kafka-operator strimzi/strimzi-kafka-operator \
    --version 0.32.0 \
    -n event \
    -f kafka-operator-values.yaml \
    > kafka-operator.yaml
```

```shell
helm upgrade kafka-operator strimzi/strimzi-kafka-operator \
    --install \
    --version 0.32.0 \
    -n event \
    --history-max 3 \
    -f kafka-operator-values.yaml
```
