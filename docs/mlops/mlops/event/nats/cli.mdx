## 설치

```shell
yay -S nats-cli
```

## Context

```shell
nats context add <context> [<flags>]
```

- `<flags>`
  - `--server <host>:<port>`
  - `--description <description>`

```shell
nats context ls
```

```shell
nats context select [<context>]
```

```shell
nats context rm <context>
```

## Stream

```shell
nats stream add [<stream>] [<flags>]
```

- `<flags>`
  - `--config=<configJson>`: JSON 형식의 설정 파일 사용

```shell
nats stream ls
```

```shell
nats stream report [<flags>]
```

- `<flags>`
  - `--name`: Stream 이름 순으로 정렬하여 표시

```shell
nats stream view [<stream>]
```

```shell
nats stream rm [<stream>]
```

```shell
nats stream purge [<stream>]
```

## Consumer

```shell
nats consumer add [<stream>] [<consumer>] [<flags>]
```

- `<flags>`
  - `--config=<configJson>`: JSON 형식의 설정 파일 사용

```shell
nats consumer ls [<stream>]
```
