## Authorization(User)

- https://docs.nats.io/running-a-nats-service/configuration/securing_nats/auth_intro

```yaml title="nats-values.yaml"
config:
  merge:
    authorization:
      users:
        - user: <user>
          password: <password>
          # nkey: <nkey>
          # permissions: {}
```

### API permissions

- https://docs.nats.io/reference/reference-protocols/nats_api_reference

```yaml title="nats-values.yaml"
config:
  merge:
    accounts:
      <account>:
        users:
          - user: <user>
            permissions:
              publish:
                deny:
                  - <apiSubject> # API 중 막고 싶은 것을 적으면 됩니다.
      SYS:
        users:
          - user: << $NATS_ADMIN_USER >>
            password: << $NATS_ADMIN_PASSWORD >>

    system_account: SYS
```
