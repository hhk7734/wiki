## Installation

```shell
helm repo add aws-cloud-controller-manager https://kubernetes.github.io/cloud-provider-aws \
&& helm repo update aws-cloud-controller-manager
```

```shell
mkdir -p aws/aws-cloud-controller-manager/base
```

```shell
helm search repo aws-cloud-controller-manager -l
```

```shell
helm show values aws-cloud-controller-manager/aws-cloud-controller-manager \
    --version 0.0.4 \
    > aws/aws-cloud-controller-manager/base/values.yaml
```

```yaml title="aws/aws-cloud-controller-manager/base/values.yaml"
# ...

args:
  - --v=2
  - --cloud-provider=aws
  - --configure-cloud-routes=false
# ...
```

```shell
helm upgrade aws-cloud-controller-manager aws-cloud-controller-manager/aws-cloud-controller-manager \
    --install \
    --version 0.0.4 \
    -f aws/aws-cloud-controller-manager/base/values.yaml
```

```shell
helm get manifest aws-cloud-controller-manager \
    > aws/aws-cloud-controller-manager/base/manifest.yaml
```
