## Sink 설정 방법

```yaml
sinks:
  <sinkID>:
    inputs:
      - <sourceID|transformID>
    type: <sourceType>
    <option>: <value>
```

## Loki

- https://vector.dev/docs/reference/configuration/sinks/loki/

```yaml
sinks:
  loki:
    inputs: []
    type: loki
    endpoint: "http://loki-distributor:3100"
    encoding:
      codec: json
    labels:
      job: vector
```
