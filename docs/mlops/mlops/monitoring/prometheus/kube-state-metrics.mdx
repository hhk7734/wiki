## kube-state-metrics

Kubernetes API를 사용하여 객체의 상태를 수집하는 서비스입니다. `kubectl` 명령어를 통해 얻는 정보를 모니터링 서비스에서 사용할 수 있도록 제공한다고 생각하면 됩니다.

## Metrics

- https://github.com/kubernetes/kube-state-metrics/tree/master/docs

<br />

- CPU
  - `kube_pod_container_resource_requests{resource="cpu"} <requests.cpu>`
  - `kube_pod_container_resource_limits{resource="cpu"} <limits.cpu>`
- Memory
  - `kube_pod_container_resource_requests{resource="memory"} <requests.memory>`
  - `kube_pod_container_resource_limits{resource="memory"} <limits.memory>`
