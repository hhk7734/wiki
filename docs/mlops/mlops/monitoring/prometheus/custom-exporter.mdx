## metric 이름

`<namespace>[_<subsystem>]_<metric>`의 형식이 많이 사용됩니다.

- `container_cpu_usage_seconds_total`
  - namespace: `container`
  - subsystem: `cpu`
  - metric: `usage_seconds_total`
- `container_memory_working_set_bytes`
  - namespace: `container`
  - subsystem: `memory`
  - metric: `working_set_bytes`
