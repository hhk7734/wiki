## metric 이름

`<namespace>_<metric>[_<unit>]`의 형식을 주로 사용하는 것 같습니다.

- `container_cpu_usage_seconds_total`
  - namespace: `container`
  - metric: `cpu_usage`
  - unit: `seconds_total`
- `container_memory_working_set_bytes`
  - namespace: `container`
  - metric: `memory_working_set`
  - unit: `bytes`
