## 설치

```shell
helm repo add grafana https://grafana.github.io/helm-charts
```

```shell
helm repo update grafana \
&& helm search repo grafana/grafana -l | head -n 10
```

```shell
helm pull grafana/grafana --version 8.6.4
```

```shell
helm show values grafana/grafana --version 8.6.4 > grafana-8.6.4.yaml
```

```yaml title="grafana-values.yaml"
extraLabels: {}

podLabels: {}

resources: {}

tolerations: []
```

```shell
helm template grafana grafana-8.6.4.tgz \
    -n grafana \
    -f grafana-values.yaml \
    > grafana.yaml
```

```shell
helm upgrade grafana grafana-8.6.4.tgz \
    --install \
    --history-max 5 \
    -n grafana \
    -f grafana-values.yaml
```
