## 설치

- https://github.com/rancher/local-path-provisioner

```shell
git clone https://github.com/rancher/local-path-provisioner.git \
    --branch v0.0.25 \
    --single-branch
```

```shell
helm package ./local-path-provisioner/deploy/chart/local-path-provisioner
```

```shell
helm show values local-path-provisioner-0.0.25.tgz \
    > local-path-provisioner-0.0.25.yaml
```

```yaml title="local-path-provisioner-values.yaml"
storageClass:
  defaultClass: false # storageclass.kubernetes.io/is-default-class=false

affinity: {}

tolerations: []
```

```shell
helm template local-path-provisioner local-path-provisioner-0.0.25.tgz \
    -n storage \
    -f local-path-provisioner-values.yaml \
    > local-path-provisioner.yaml
```

```shell
helm upgrade local-path-provisioner local-path-provisioner-0.0.25.tgz \
    --install \
    --history-max 3 \
    -n storage \
    -f local-path-provisioner-values.yaml
```
