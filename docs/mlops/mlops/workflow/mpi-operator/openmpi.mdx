## 설치

:::info[Reference]

- [OpenMPI / Download](https://www-lb.open-mpi.org/software/ompi)

:::

```shell
tar xf openmpi-5.0.7.tar.bz2
```

```shell
cd openmpi-5.0.7
```

```shell
./configure --prefix=/usr <flags>
```

```shell
make -j $(nproc) all
```

```shell
make install
```
