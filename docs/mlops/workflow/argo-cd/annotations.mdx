## 개별 리소스에 적용

개별 리소스에 대한 동기화 옵션은 annotations를 통해 지정할 수 있습니다.

```yaml
metadata:
  annotations:
    argocd.argoproj.io/sync-options: Prune=false
    argocd.argoproj.io/compare-options: IgnoreExtraneous
```

## sync-options

- https://argo-cd.readthedocs.io/en/stable/user-guide/sync-options/

<br />

- `argocd.argoproj.io/sync-options: <option>`
  - `Prune=false`: Git에서 삭제된 리소스를 클러스터에서 삭제하지 않습니다.
  - `Delete=false`: Application이 삭제되어도 리소스를 삭제하지 않습니다.

## compare-options

- https://argo-cd.readthedocs.io/en/stable/user-guide/compare-options/

<br />

- `argocd.argoproj.io/compare-options: <option>`
  - `IgnoreExtraneous`: 해당 리소스의 동기화 상태를 무시합니다.
