## 설치

:::info[Reference]

- [local-path-provisioner GitHub](https://github.com/rancher/local-path-provisioner)

:::

:::info

```shell
git clone https://github.com/rancher/local-path-provisioner.git \
    --branch v0.0.32
```

```shell
helm package ./local-path-provisioner/deploy/chart/local-path-provisioner
```

:::

```shell
helm repo add lol-iot https://lol-iot.github.io/helm-charts
```

```shell
helm repo update lol-iot \
&& helm search repo lol-iot/local-path-provisioner -l | head -n 10
```

```shell
helm pull lol-iot/local-path-provisioner --version 0.0.32
```

```shell
helm show values lol-iot/local-path-provisioner --version 0.0.32 \
  > local-path-provisioner-0.0.32.yaml
```

```yaml title="local-path-provisioner-values.yaml"
storageClass:
  defaultClass: false # storageclass.kubernetes.io/is-default-class=false

affinity: {}

tolerations: []
```

```shell
helm template local-path-provisioner lol-iot/local-path-provisioner \
    --version 0.0.32 \
    -n kube-system \
    -f local-path-provisioner-values.yaml \
    > local-path-provisioner.yaml
```

```shell
helm upgrade local-path-provisioner lol-iot/local-path-provisioner \
    --install \
    --history-max 5 \
    --version 0.0.32 \
    -n kube-system \
    -f local-path-provisioner-values.yaml
```
