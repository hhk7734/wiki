## Gateway

:::info[Reference]

- [Gateway API / API specification / Standard # Gateway](https://gateway-api.sigs.k8s.io/reference/spec/#gateway)

:::

```yaml
apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
spec:
  gatewayClassName: <gatewayClassName>
  listeners:
    - name: http
      protocol: HTTP
      port: 80
      allowedRoutes:
        namespaces:
          from: Same
    - name: https
      protocol: HTTPS
      port: 443
      tls:
        name: Terminate
        certificateRefs:
          - name: <secretName>
      allowedRoutes:
        namespaces:
          from: Same
```

- `gatewayClassName: <name>`
- `listeners: []`
- `infrastructure`
  - `labels`
  - `annotations`
  - `parametersRef`
    - GatewayClass가 인식할 수 있는 리소스를 설정하면 Gateway에 의해 생성되는 리소스를 정의할 수 있습니다.
    - `group`
    - `kind`
    - `name`
