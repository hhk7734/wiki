## BootstrapProvider

```yaml
---
apiVersion: v1
kind: Namespace
metadata:
  name: capi-k0smotron-bootstrap

---
apiVersion: operator.cluster.x-k8s.io/v1alpha2
kind: BootstrapProvider
metadata:
  name: k0sproject-k0smotron
  namespace: capi-k0smotron-bootstrap
spec:
  version: v1.6.2
```

- `version: <version>`
  - [k0smotron GitHub / Releases](https://github.com/k0sproject/k0smotron/releases)

## K0sWorkerConfig

:::info[Reference]

- [k0smotron Docs / Reference / bootstrap # K0sWorkerConfig.spec](https://docs.k0smotron.io/stable/resource-reference/bootstrap.cluster.x-k8s.io-v1beta1/#k0sworkerconfigspec)

:::

```yaml
apiVersion: bootstrap.cluster.x-k8s.io/v1beta1
kind: K0sWorkerConfig
spec:
  version: v1.32.6+k0s.0
  useSystemHostname: true
```

- `version: <version>`
  - [k0s GitHub / Releases](https://github.com/k0sproject/k0s/releases)
- `useSystemHostname: false`
  - `true`로 설정한 경우 Node 이름이 `hostname` 명령어의 출력값으로 설정됩니다.
  - `false`로 설정한 경우 Machine의 이름으로 설정됩니다.
