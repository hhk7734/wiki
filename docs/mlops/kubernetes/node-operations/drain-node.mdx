## Node 목록 확인

```shell
kubectl get nodes
```

## 스케줄 불가로 설정(SchedulingDisabled)

```shell
kubectl cordon <nodeName>
```

## Node에 있는 Pod 비우기

```shell
kubectl drain <nodeName> --ignore-daemonsets
```

## Node 제거

```shell
kubectl delete node <nodeName>
```
