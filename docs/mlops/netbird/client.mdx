## Linux

:::info[References]

- [NetBird Docs / Get Started / Install NetBird / Linux](https://docs.netbird.io/how-to/installation/linux)

:::

```shell
curl -fsSL https://pkgs.netbird.io/install.sh | NETBIRD_RELEASE=v0.59.5 sh
```

```shell
netbird up [<flag> ...]
```

- `-m|--management-url <http|https>://<host>:<port>`
- `-k|--setup-key <key>`
