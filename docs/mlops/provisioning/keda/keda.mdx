## 설치

```shell
helm repo add kedacore https://kedacore.github.io/charts
```

```shell
helm repo update kedacore \
&& helm search repo kedacore/keda -l | head -n 10
```

```shell
helm pull kedacore/keda --version 2.18.0
```

```shell
helm show values kedacore/keda --version 2.18.0 > keda-2.18.0.yaml
```

```yaml title="keda-values.yaml"
additionalLabels: {}
```

```shell
helm template keda kedacore/keda \
    --version 2.18.0 \
    -n keda \
    -f keda-values.yaml \
    > keda.yaml
```

```shell
helm upgrade keda kedacore/keda \
    --install \
    --history-max 5 \
    --version 2.18.0 \
    -n keda \
    -f keda-values.yaml
```

## 삭제

```shell
helm uninstall -n keda keda
```

```shell
kubectl get crd -oname | grep --color=never 'keda.sh' | xargs kubectl delete
```
