## Log Retention

:::info[Reference]

- [Loki Docs / Configuration parameters # compactor](https://grafana.com/docs/loki/latest/configure/#compactor)

:::

```yaml title="loki-distributed-values.yaml"
loki:
  compactor:
    retention_enabled: true
    delete_request_store: s3

  limits_config:
    retention_period: 4320h
```
