## Data Source

:::info[Reference]

- [Grafana / Provision Grafana / Data Sources](https://grafana.com/docs/grafana/latest/administration/provisioning/#data-sources)

:::

```yaml title="grafana-values.yaml"
datasources:
  datasources.yaml:
    apiVersion: 1
    deleteDatasources: []
    datasources: []
```

- `datasources: []`
  - `name: <name>`
  - `type: <type>`
    - prometheus, loki, ...
  - `url: <url>`
  - `isDefault: <bool>`
  - `uid: <uid>`: 해당 데이터 소스를 다른 설정에서 참조할 때 사용되는 고유 식별자를 설정합니다.

## Prometheus

```yaml title="grafana-values.yaml"
datasources:
  datasources.yaml:
    datasources:
      - name: prometheus
        type: prometheus
        url: http://prometheus-operated.prometheus-stack.svc.cluster.local:9090
        isDefault: true
        uid: 194258a3-be9e-407a-a922-f47535e102d3
```

## Loki

```yaml title="grafana-values.yaml"
datasources:
  datasources.yaml:
    datasources:
      - name: loki
        type: loki
        url: http://loki-query-frontend.loki.svc.cluster.local:3100
        isDefault: false
        uid: 467772c5-1416-4599-967a-844d3afc17f5
```

## Tempo

:::info[Reference]

- [Grafana / Configure a Tempo data source](https://grafana.com/docs/grafana/latest/datasources/tempo/configure-tempo-data-source/)

:::

```yaml title="grafana-values.yaml"
datasources:
  datasources.yaml:
    datasources:
      - name: tempo
        type: tempo
        url: http://tempo-query-frontend.tempo.svc.cluster.local:3100
        isDefault: false
        uid: ce162960-6979-473c-a6c5-9240622afe8c
        basicAuth: false
```
