## Semantic Conventions

:::info[References]

- [OpenTelemetry Semantic Conventions](https://opentelemetry.io/docs/specs/semconv/)

:::

OpenTelemetry Semantic Conventions은 수집하는 데이터에 정의할 수 있는 표준화된 Attributes 집합을 제공합니다. 이를 통해 다양한 시스템과 서비스 간의 일관된 데이터 표현이 가능해집니다.
