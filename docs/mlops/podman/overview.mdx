## Podman, Buildah, Skopeo

- Podman
  - Docker daemon 없이 컨테이너를 실행하고 관리하는 도구입니다.
  - rootless 실행과 `containers/*` 설정 파일을 적극적으로 사용합니다.
- Buildah
  - 컨테이너 이미지를 빌드하는 데 초점을 둔 도구입니다.
  - Dockerfile 기반 빌드와 스크립트 기반 빌드를 모두 지원합니다.
- Skopeo
  - 레지스트리 간 이미지 복사, inspect, 로그인 작업에 특화된 도구입니다.
  - 로컬 컨테이너 런타임 없이도 이미지 메타데이터를 다룰 수 있습니다.
