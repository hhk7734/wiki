## Cilium Gateway API 설정

:::info[Reference]

- [Cilium / Gateway API Support](https://docs.cilium.io/en/stable/network/servicemesh/gateway-api/gateway-api/)

:::

Cilium에서 Gateway API를 활성화하기 전에 Gateway API CRD가 먼저 설치되어 있어야합니다.

:::warning

Cilium v1.16.4 기준 Gateway API v1.1.0을 지원합니다.

:::

```yaml title="cilium-values.yaml"
gatewayAPI:
  enabled: true

l7Proxy: true

nodePort:
  enabled: true

kubeProxyReplacemen: "true"
k8sServiceHost: "auto"
k8sServicePort: "auto"
```

cilium-operator와 cilium\(agent\)을 재시작해야합니다.
