### Helm으로 Cilium 설치하기

:::info[Reference]

- [Cilium / Installation using Helm](https://docs.cilium.io/en/stable/installation/k8s-install-helm/)

:::

```shell
helm repo add cilium https://helm.cilium.io/
```

```shell
helm repo update cilium \
&& helm search repo cilium/cilium -l | head -n 10
```

```shell
helm pull cilium/cilium --version 1.18.2
```

```shell
helm show values cilium/cilium --version 1.18.2 > cilium-1.18.2.yaml
```

```yaml tilte="cilium-values.yaml"
ipam:
  # -- Configure IP Address Management mode.
  # ref: https://docs.cilium.io/en/stable/network/concepts/ipam/
  mode: "cluster-pool"
  operator:
    # -- IPv4 podCIDR list
    clusterPoolIPv4PodCIDRList: ["10.244.0.0/16"]
    # -- IPv4 CIDR mask size to delegate to individual nodes for IPAM.
    clusterPoolIPv4MaskSize: 25
    # -- IPv6 podCIDR list
    clusterPoolIPv6PodCIDRList: ["fd00::/104"]
    # -- IPv6 CIDR mask size to delegate to individual nodes for IPAM.
    clusterPoolIPv6MaskSize: 120

operator:
  replicas: 1
```

```shell
helm template cilium cilium/cilium \
    --version 1.18.2 \
    -n kube-system \
    -f cilium-values.yaml \
    > cilium.yaml
```

```shell
helm upgrade -i cilium cilium/cilium \
    --history-max 5 \
    --version 1.18.2 \
    -n kube-system \
    -f cilium-values.yaml
```

```shell
cilium status --wait
```
