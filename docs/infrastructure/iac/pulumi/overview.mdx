Pulumi는 TypeScript, Python 같은 일반 프로그래밍 언어로 인프라를 선언하고 배포하는 IaC 도구입니다. 스택, 설정, provider 같은 개념이 분리되어 있어서 코드 재사용과 환경별 구성을 구조적으로 관리하기 좋습니다.
