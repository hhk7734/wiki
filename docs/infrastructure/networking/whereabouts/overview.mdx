`whereabouts`는 클러스터 전체에서 사용할 수 있는 IPAM CNI 플러그인으로, 여러 노드에 걸쳐 IP 대역을 중복 없이 할당해야 할 때 사용합니다.

## 설정

```yaml
apiVersion: "k8s.cni.cncf.io/v1"
kind: NetworkAttachmentDefinition
spec:
  config: |
    {
      "cniVersion": "0.3.1",
      "name": "xxx",

      // ...

      "ipam": {
          "type": "whereabouts"
      }
    }
```

- `ipam`
  - `range: <ip>/<cidr>`
  - `range_start: <ip>`
    - 설정하지 않으면 `range`로부터 추측합니다.
  - `range_end: <ip>`
    - 설정하지 않으면 `range`로부터 추측합니다.
  - `ipRanges: []`
    - `range: <ip>/<cidr>`
  - `exclude: [<ip>/<cidr>]`
  - `node_slice_size: /<cidr>`
