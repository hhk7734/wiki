AWS EBS Container Storage Interface Driver

:::info
[설치 가이드](/docs/infrastructure/storage/aws-ebs-csi-driver/install.mdx) 문서는 별도로 분리되어 있습니다.
:::

- https://github.com/kubernetes-sigs/aws-ebs-csi-driver/blob/master/docs/install.md

`aws-ebs-csi-driver`는 Amazon EBS 볼륨을 Kubernetes CSI 인터페이스로 연결해 주는 드라이버입니다. 주로 EKS에서 동적 PersistentVolume 생성과 StorageClass 기반 블록 스토리지를 제공할 때 사용합니다.
