## hostPath

- https://kubernetes.io/docs/concepts/storage/volumes/#hostpath

  volumes:
    - name: <volumeName>
      hostPath:
        path: <path>
        type: ""

## persistentVolumeClaim

```yaml

  volumes:
    - name: <volumeName>
      persistentVolumeClaim:
        claimName: <claimName>
        readOnly: false
```
