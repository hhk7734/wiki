`local-path-provisioner`는 노드의 로컬 디렉터리를 기준으로 PersistentVolume을 동적으로 생성하는 가벼운 스토리지 프로비저너입니다. 테스트 환경이나 단일 노드, 또는 공유 스토리지 없이 빠르게 PVC를 써야 하는 환경에서 주로 사용합니다.
