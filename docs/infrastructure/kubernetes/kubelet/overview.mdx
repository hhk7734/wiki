:::

```shell
kubectl get --raw /api/v1/nodes/<nodeName>/proxy/metrics
```

```shell
kubectl get --raw /api/v1/nodes/<nodeName>/proxy/metrics/cadvisor
```

```shell
kubectl get --raw /api/v1/nodes/<nodeName>/proxy/metrics/probes
```
