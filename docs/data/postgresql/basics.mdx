## 초기화

```shell
sudo su postgres
```

```shell
initdb --locale=C.UTF-8 --encoding=UTF8 -D /var/lib/postgres/data
```

## Access

### Local

```shell
sudo -u postgres psql
```

### Remote

```shell
psql -h <host> -U <user> -d <database>
```

## Help

```sql
\h [command]
```

`\h`만 입력하면 사용가능한 SQL 명령어 목록이 출력됩니다. `command`를 입력하면 해당 명령어의 설명이 출력됩니다.

```sql
\?
```

`\`로 시작하는 명령어 목록과 설명이 출력됩니다.
