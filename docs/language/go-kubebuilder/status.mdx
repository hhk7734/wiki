## Status

status는 spec에서 정의한 원하는 상태(desired state)에 대한 현재 상태(observed state)를 표현합니다. 이를 통해 Reconciler는 시스템의 실제 상태를 원하는 상태로 맞추기 위해 작업을 수행할 수 있습니다.
