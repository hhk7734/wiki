## Installation

- https://helm.sh/docs/intro/install/

```shell
curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3 | bash
```

## Repository

```shell
helm repo add <NAME> <URL> [flags]
```

Ex) `helm repo add stable https://charts.helm.sh/stable`

```shell
helm repo list [flags]
```

```shell
helm search repo <keyword> [flags]
```

- `-l`, `--versions`: 여러 버전을 표시합니다.

```shell
helm repo update [REPO1 [REPO2 ...]] [flags]
```

## Chart

### Creating Chart

```shell
helm create <NAME> [flags]
```

```shell
helm template <NAME> <CHART> [flags] > <NAME>.yaml
```

- https://helm.sh/docs/topics/charts/

### Installation

```shell
helm show values <CHART> [flags]
```

```shell
helm install <NAME> <CHART> [flags]
```

:::info
`helm upgrade <NAME> <CHART> --install [flags]` 을 사용해도 됩니다.
:::

- `-f <path>`: 오버라이드 할 YAML 설정 파일
- `--set <name>=<value>`: 오버라이드 할 설정

```shell
helm list [flags]
```

- 설치된 릴리즈의 NAME, NAMESPACE, REVISION, UPDATED, STATUS, CHART, APP VERSION 표시
- `-A | --all-namespaces`

```shell
helm status <NAME> [flags]
```

### Management

```shell
helm upgrade <NAME> <CHART> [flags]
```

- `--install`: 설치가 안되어 있으면 설치합니다.

```shell
helm get values <CHART> [flags]
```

```shell
helm get manifest <NAME> [flags]
```

```shell
helm history <NAME> [flags]
```

```shell
helm rollback <NAME> <REVISION> [flags]
```

### Uninstallation

```shell
helm uninstall <NAME> [...] [flags]
```
