## Creating template files

м•„лһҳ лӘ…л №м–ҙлҘј нҶөн•ҙ `<package>`мҷҖ лҸҷмқјн•ң л””л үн„°лҰ¬м—җ `<package>-<version>`, `<package>-<version>.tar.gz`, `<package>_<version>.orig.tar.gz`мқҙ мғқм„ұлҗ©лӢҲлӢӨ.

мғқм„ұлҗң template filesмқҖ `<package>-<version>` л””л үн„°лҰ¬м—җм„ң нҷ•мқё к°ҖлҠҘн•©лӢҲлӢӨ.

`<package>`л””л үн„°лҰ¬лЎң мқҙлҸҷ нӣ„ м•„лһҳ лӘ…л №м–ҙлҘј мӢӨн–үн•©лӢҲлӢӨ.

```shell
debmake -t -p <package> -u <version> [-r <revision>] -x[0-4]
```
