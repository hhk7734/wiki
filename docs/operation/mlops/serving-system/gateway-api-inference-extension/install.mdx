## 설치

```shell
helm repo add lol-iot https://lol-iot.github.io/helm-charts
```

```shell
helm repo update lol-iot \
&& helm search repo lol-iot/gateway-api-inference-extension -l | head -n 10
```

```shell
helm pull lol-iot/gateway-api-inference-extension --version 0.5.1
```

```shell
helm show values lol-iot/gateway-api-inference-extension --version 0.5.1 \
  > gateway-api-inference-extension-0.5.1.yaml
```

```shell
helm template gateway-api-inference-extension lol-iot/gateway-api-inference-extension \
  --version 0.5.1 \
  -n kube-system \
  > gateway-api-inference-extension.yaml
```

```shell
helm upgrade -i gateway-api-inference-extension lol-iot/gateway-api-inference-extension \
    --history-max 5 \
    --version 0.5.1 \
    -n kube-system
```
