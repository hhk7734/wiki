Terraform은 선언형 `.tf` 파일과 provider 생태계를 중심으로 인프라를 관리하는 대표적인 IaC 도구입니다. backend, state, module, provider를 기준으로 워크플로우를 구성하며, plan/apply 기반의 변경 관리가 핵심입니다.
