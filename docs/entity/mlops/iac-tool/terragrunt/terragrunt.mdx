Terragrunt는 Terraform 모듈에 공통 backend, provider, include 규칙을 덧씌워서 중복을 줄이고 환경별 구성을 정리하는 도구입니다. 여러 Terraform 모듈을 계층적으로 관리해야 할 때 특히 유용합니다.
