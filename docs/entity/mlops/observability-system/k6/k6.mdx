`k6`는 HTTP API, gRPC, WebSocket 같은 워크로드를 스크립트 기반으로 부하 테스트할 수 있는 도구입니다. 로컬 실행부터 확장 프로그램 기반 커스텀 빌드까지 폭넓게 사용할 수 있습니다.
