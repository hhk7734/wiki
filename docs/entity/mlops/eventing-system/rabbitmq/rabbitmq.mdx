## CRDs

### RabbitmqCluster

- https://www.rabbitmq.com/kubernetes/operator/using-operator.html

```yaml
apiVersion: rabbitmq.com/v1beta1
kind: RabbitmqCluster
metadata:
  name: rabbitmq
  namespace: workflow
spec:
  replicas: 1
  rabbitmq:
    additionalConfig: |
      cluster_partition_handling = pause_minority
      vm_memory_high_watermark_paging_ratio = 0.99
      disk_free_limit.relative = 2.0
      collect_statistics_interval = 30000
  resources:
    requests:
      cpu: 200m
      memory: 2Gi
    limits:
      memory: 2Gi
  persistence:
    storage: 10Gi
  affinity:
    podAntiAffinity:
      requiredDuringSchedulingIgnoredDuringExecution:
        - labelSelector:
            matchExpressions:
              - key: app.kubernetes.io/name
                operator: In
                values:
                  - rabbitmq
          topologyKey: kubernetes.io/hostname
  override:
    statefulSet:
      spec:
        template:
          spec:
            containers: []
            topologySpreadConstraints:
              - maxSkew: 1
                topologyKey: "topology.kubernetes.io/zone"
                whenUnsatisfiable: DoNotSchedule
                labelSelector:
                  matchLabels:
                    app.kubernetes.io/name: rabbitmq
```

### Policy

- https://github.com/rabbitmq/messaging-topology-operator/blob/main/api/v1beta1/policy_types.go

```yaml

  name: <crd-name>
  namespace: workflow

  name: <policy-name>
  vhost: /
  pattern: <pattern> # мқҙ PolicyлҘј м Ғмҡ©н•  Queue лҳҗлҠ” Exchangeмқҳ мқҙлҰ„ нҢЁн„ҙ

    <key>: <value>
  rabbitmqClusterReference: # Policyк°Җ л“ұлЎқлҗ  нҒҙлҹ¬мҠӨн„°
    name: rabbitmq
```
