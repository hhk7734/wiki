AWS EFS Container Storage Interface Driver

:::info
[설치 가이드](/docs/operation/data/storage-interface/aws-efs-csi-driver/install.mdx) 문서는 별도로 분리되어 있습니다.
:::

- https://github.com/kubernetes-sigs/aws-efs-csi-driver

`aws-efs-csi-driver`는 Amazon EFS 파일 시스템을 Kubernetes CSI 인터페이스로 연결해 주는 드라이버입니다. 여러 Pod에서 공유 파일시스템을 써야 할 때 주로 사용합니다.
