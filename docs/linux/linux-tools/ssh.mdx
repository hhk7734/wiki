## 키 생성

```shell
ssh-keygen -t ed25519 -N "" -C "your_email@example.com" -f <path>
```

```shell
ssh-keygen -t rsa -b 2048 -N "" -C "your_email@example.com" -f <path>
```

## config

- https://man.openbsd.org/ssh_config

`~/.ssh/config` 또는 `/etc/ssh/ssh_config` 파일에 아래 설정을 추가할 수 있습니다.

```shell
Host <alias>
  HostName <host>
  User <user>
  PreferredAuthentications publickey
  IdentityFile <path>
  AddKeysToAgent yes
```

## ssh agent 실행

```shell
eval "$(ssh-agent -s)"
```

## 키 등록

```shell
ssh-add ~/.ssh/<id file>
```

## 퍼블릭 키 접속할 곳에 등록

```shell
ssh-copy-id -i ~/.ssh/<id file>.pub <remote>
```

## SCP

source를 target으로 보낼 수 있는 명령어입니다.

원격지의 경우 **\<user\>@\<ip\>:\<path\>** 로 구성됩니다.

```shell
scp [options] <source> <target>
```

## 파일 권한

```shell
sudo chmod 700 ~/.ssh
sudo chmod 600 ~/.ssh/*
```

:::warning
파일 접근 권한은 로컬과 원격 모두 설정되어 있어야 합니다.
:::

## SSH Tunnel

```shell title="/etc/ssh/sshd_config"
AllowTcpForwarding yes
PermitTunnel yes
```

```shell
ssh -f -N -L <localPort>:<host>:<port> -p <tunnelPort> <tunnelUser>@<tunnelHost>
```

```shell
lsof -i4 -P | grep -i "listen" | grep <localPort>
```

```shell
nc -zv 127.0.0.1 <localPort>
```

```shell
ps -lef | grep ssh | grep "\-L \?<localPort>" | awk '{print $4}' | xargs kill -9
```
