## Create patch files

```shell
git format-patch [-<num of commit>] [commit]
```

## Apply patch file

```shell
git am <patch>
```

### Fix patch errors

```shell
git apply <patch> --reject
```

Look at the created **\<.ref\>** file and fix the errors.

```shell
git add <modified files>
```

```shell
git am --continue
```
