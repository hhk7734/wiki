- NewContext: 이전 컨텍스트가 있는 경우 이전 컨텍스트를 복사해서 컨텍스트를 만들고, 없으면 새로운 컨텍스트를 만든 후에 인자로 받은 필드를 컨텍스트에 추가 또는 업데이트합니다.
- SetFields: 컨텍스트에 필드를 추가 또는 업데이트합니다.
- GetField: 컨텍스트에서 키에 해당하는 필드를 찾아서 반환합니다. 예를 들어 `(value, isExist)`를 반환하여 존재 유무와 값을 반환합니다.
- GetFields
- DeleteField: 컨텍스트에서 키에 해당하는 필드를 찾아서 삭제합니다.
