Labview를 실행하면 아래와 같은 창이 열립니다. myRIO 툴킷을 설치한 경우 myRIO라고 표시됩니다.

<center>
	<img src={useBaseUrl("img/lang/labview/labview-project-start.png")} />
</center>

create project를 누르면 아래와 같은 창이 열립니다. 새 프로젝트를 선택합니다.

<center>
	<img src={useBaseUrl("img/lang/labview/labview-project-templete.png")} />
</center>

새 프로젝트를 생성하면 프로젝트 탐색기가 열립니다. **프로젝트 탐색기에는 프로그램이 실행 될 장치와 해당 장치에서 실행되 는 VI 목록이 표시됩니다.**

예를 들어 myRIO 두 대, 컴퓨터 두 대를 사용하여 동시에 실행되어야 하는 프로젝트가 있다고 하면 프로젝트 하위 구조에 myRIO 두 대, 컴퓨터 두 대가 표시됩니다.

코드를 컴퓨터에서 실행시킬 때는 아래와 같이 내 컴퓨터에 VI를 추가하면 됩니다.

<center>
	<img src={useBaseUrl("img/lang/labview/labview-create-vi.gif")} />
</center>
