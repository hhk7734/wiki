## 설치

- [asdf로 런타임 버전 관리하기](/docs/lang/etc/command-line-tools/asdf#플러그인)

```shell
asdf plugin add rust
```

## Setup

```shell
cargo new <project>
```

## Build

```shell
cargo build [flags]
```

- `[flags]`
  - `--release`: 최적화된 빌드
