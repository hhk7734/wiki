## 설치

```shell
asdf plugin add rust
```

```shell
asdf list all rust
```

```shell
asdf install rust 1.68.2
```

```shell
asdf global rust 1.68.2
```

```shell
asdf local rust 1.68.2
```

```shell
asdf current rust
```

## Setup

```shell
cargo new <project>
```

## Build

```shell
cargo build [flags]
```

- `[flags]`
  - `--release`: 최적화된 빌드
