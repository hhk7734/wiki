## patch files 생성

```shell
git format-patch [-<num of commit>] [commit]
```

## patch file 적용

```shell
git am <patch>
```

### patch 중 에러 수정

```shell
git apply <patch> --reject
```

생성된 **\<.ref\>** 파일을 보고 에러를 수정합니다.

```shell
git add <modified files>
```

```shell
git am --continue
```
