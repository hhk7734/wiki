## 원하는 저장소 fork

- 원하는 저장소에가서 fork
  `https://github.com/<owner>/<repository>`
- fork된 저장소
  `https://github.com/<contributor>/<repository>`

## fork된 저장소 clone

```shell
git clone https://github.com/<contributor>/<repository>
```

## branch 관리

### branch 확인

```shell
git branch -a
```

### branch 생성

```shell
git switch -c <workingBranch>
```

### branch 변경

```shell
git switch <workingBranch>
```

## code 수정

```shell
git add <file>
```

```shell
git commit -m "<message>"
```

```shell
git push
```

## 원 저장소에 업데이트된 내용을 local에 병합하기

```shell
git remote add upstream https://github.com/<owner>/<repository>
```

```shell
git pull upstream
```

```shell
git switch <workingBranch>
```

```shell
git rebase upstream/<mainBranch>
```

### conflict 파일이 있는 경우

```shell
vim <conflictFile>
```

```shell
git add <conflictFile>
```

```shell
git commit
```

```shell
git push
```
