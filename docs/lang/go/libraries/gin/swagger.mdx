## Canonical example

```go title="main.go"
package main

import (
	"net/http"

	docs "github.com/HITS-AI/one_scenario_submit_api/docs"
	"github.com/gin-gonic/gin"
	swaggerfiles "github.com/swaggo/files"
	ginSwagger "github.com/swaggo/gin-swagger"
)

// @BasePath /api/v1

// PingExample godoc
// @Summary  ping example
// @Schemes
// @Description  do ping
// @Tags         example
// @Accept       json
// @Produce      json
// @Success      200  {string}  Helloworld
// @Router       /example/helloworld [get]
func Helloworld(g *gin.Context) {
	g.JSON(http.StatusOK, "helloworld")
}

func main() {
	r := gin.Default()
	docs.SwaggerInfo_swagger.BasePath = "/api/v1"
	v1 := r.Group("/api/v1")
	{
		eg := v1.Group("/example")
		{
			eg.GET("/helloworld", Helloworld)
		}
	}
	r.GET("/swagger/*any", ginSwagger.WrapHandler(swaggerfiles.Handler))
	r.Run(":8080")
}
```

```shell
swag fmt && swag init
```

```shell
go run .
```

- http://localhost:8080/swagger/index.html

## Comments 형식

### General API Info

- https://github.com/swaggo/swag#general-api-info

### API Operation

- https://github.com/swaggo/swag#api-operation

#### @Param

```go
// @Param <param_name> <param_type> <data_type> <mandatory> <comment> [<attribute>]
```

Ex) `// @Param user_id path int true "User ID"`

- `<param_type>`
  - query
  - path
  - header
  - body
  - fromData
- `<data_type>`
  - string
  - integer
  - number
  - boolean
  - 사용자가 정의한 struct
- `<attirbute>`
  - https://github.com/swaggo/swag#attribute

:::info
data_type으로 사용자 정의 struct를 사용할 때, attribute를 struct의 태그로 사용할 수 있습니다.
:::

#### @Router

```go
// @Router /<path> [<httpMethod>]
```

Ex) `// @Router /examples/user/{user_id}/address [put]`

#### @Success, @Failure, @Response

```go
// @Success <status> {<param type>} <data type> <comment>
```
