## 설치

- https://github.com/go-yaml/yaml

```shell
go get -u gopkg.in/yaml.v3
```

## Yaml 파일 읽기

```go
package main

import (
	"fmt"
	"os"

	"gopkg.in/yaml.v3"
)

func main() {
	text, err := os.ReadFile("test.yaml")
	if err != nil {
		panic(err)
	}

	var data interface{}
	err = yaml.Unmarshal(text, &data)
	if err != nil {
		panic(err)
	}

	fmt.Println(data)
}
```
