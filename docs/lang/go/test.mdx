## Test

```shell
go test [<flag>] [<pathPattern>]
```

- `<flag>`
  - `-v`: Verbose
  - `-coverprofile <file>`: 커버리지 프로파일을 텍스트 파일로 저장합니다.
- `<pathPattern>`
  - `./...`: 현재 디렉토리의 모든 하위 디렉토리를 포함합니다.

## Table Driven Tests

- https://github.com/golang/go/wiki/TableDrivenTests

```go
func TestAddOne(t *testing.T) {
	cases := []struct {
		in   int
		want int
	}{
		{1, 2},
		{2, 3},
		{3, 4},
	}

	for _, c := range cases {
		got := c.in + 1
		assert.Equal(t, got, c.want)
	}
}
```
