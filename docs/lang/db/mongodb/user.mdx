## User

### User 생성

- [createUser](https://www.mongodb.com/docs/manual/reference/method/db.createUser/)

```js
// use <database>

db.createUser({
	user: "<user>",
	pwd: "<password>", // or passwordPrompt()
	roles: [
		// "<role>",
		{ role: "<role>", db: "<database>" },
	],
});
```
