## Access

### Local

```shell
sudo -u postgres psql
```

### Remote

```shell
psql -h <host> -U <user>
```

## Help

```sql
\h [command]
```

`\h`만 입력하면 사용가능한 SQL 명령어 목록이 출력됩니다. `command`를 입력하면 해당 명령어의 설명이 출력됩니다.

```sql
\?
```

`\`로 시작하는 명령어 목록과 설명이 출력됩니다.

## Role(User)

```sql
CREATE USER <user> WITH PASSWORD '<password>';
```

## Database

```sql
CREATE DATABASE <database_name> with OWNER <user>;
```

```sql
\list+
```

```sql
\connect <database_name>;
```

```sql
DROP DATABASE <database_name>;
```

## Schema

## Table

### 생성, 삭제

```sql
CREATE TABLE <table_name> (
    <column_name> <data_type> [options]
    [, <column_name> <data_type> [options]]
);
```

```sql
\d[+] <table_name>
```

```sql
TRUNCATE TABLE <table_name>;
```

```sql
DROP TABLE <table_name>;
```

### Column

```sql
ALTER TABLE <table_name>
ADD COLUMN <column_name> <data_type> [options];
```

```sql
ALTER TABLE <table_name>
DROP COLUMN <column_name>;
```

### Row

```sql
INSERT INTO <table_name> (<column_name>[, <column_name>])
VALUES (<value>[, <value>,]);
```

```sql
SELECT <column_name>[,
    <column_name>]
FROM <table_name>
[WHERE <condition>];
```

```sql
UPDATE <table_name>
SET <column_name> = <value>[,
    <column_name> = <value>]
[WHERE <condition>];
```

```sql
DELETE FROM <table_name>
WHERE <condition>;
```
