## [INNER] JOIN

```sql
SELECT
    <select_expr>
FROM
    <table_1> AS <alias_1>
    [INNER] JOIN <table_2> AS <alias_2> ON <join_condition>
```

## LEFT|RIGTH|OUTER JOIN

```sql
SELECT
    <select_expr>
FROM
    <table_1> AS <alias_1>
    LEFT|RIGHT|OUTER JOIN <table_2> AS <alias_2> ON <join_condition>
```

## CROSS JOIN

```sql
SELECT
    <select_expr>
FROM
    <table_1> AS <alias_1>
    CROSS JOIN <table_2> AS <alias_2>
```
