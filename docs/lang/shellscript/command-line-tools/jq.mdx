## 설치

```shell
sudo pacman -S jq
```

## 실시간 json log 파싱

```shell
<program> | jq -R -r '. as $line | try fromjson catch $line'
```

```shell
alias log2jq="jq -R -r '. as \$line | try fromjson catch \$line'"
```
