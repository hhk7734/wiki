## 설치

```shell
asdf plugin add nodejs
```

```shell
asdf list all nodejs
```

```shell
asdf install nodejs 18.12.1
```

```shell
asdf global nodejs 18.12.1
```

```shell
asdf local nodejs 18.12.1
```

```shell
asdf current nodejs
```

:::info
`.nvmrc`를 사용하고 싶다면 `$HOME/.asdfrc`에 `legacy_version_file = yes`를 추가하면 됩니다.
:::
