## 설정

- [Prisma Schema Overview](https://www.prisma.io/docs/orm/prisma-schema/overview)

```hcl title="prisma/schema.prisma"
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "<provider>"
  url      = env("DATABASE_URL")
}
```

- [Connection URLs](https://www.prisma.io/docs/orm/reference/connection-urls)
- [Databases](https://www.prisma.io/docs/orm/overview/databases)

```shell title=".env"
DATABASE_URL=<provider>://<user>:<password>@<host>:<port>/<database>?<option>
```
