## ResNet

**Res**: Residual

### ResBlock

```ini

batch_normalize=1
filters=32
size=1
stride=1
pad=1
activation=mish

[convolutional]
batch_normalize=1
filters=64
size=3
stride=1
pad=1
activation=mish

[shortcut]
from=-3
activation=linear
```

## CSPResNet

**CSP**: Cross Stage Partial

```ini
[convolutional]
batch_normalize=1
filters=64
size=1
stride=1
pad=1
activation=mish

[route]
layers = -2

[convolutional]
batch_normalize=1
filters=64
size=1
stride=1
pad=1
activation=mish

... resnet -> 3 x n layers ...

[convolutional]
batch_normalize=1
filters=64
size=1
stride=1
pad=1
activation=mish

[route]
layers = -1,-7 # -(3 x n + 4)

[convolutional]
batch_normalize=1
filters=64
size=1
stride=1
pad=1
activation=mish
```
