## Timestamp

```go
from google.protobuf.timestamp_pb2 import Timestamp

timestamp = Timestamp()
timestamp.GetCurrentTime()
// timestamp.FromDatetime(datetime.datetime.now())

t = timestamp.ToDatetime() // datetime.datetime
```
