## try-except

```python
try:
    ...
except Exception as e:
    ...
else:
    ...
finally:
    ...
```

| 정상 |         | 비정상 |
| :--: | :-----: | :----: |
|  ✓   |   try   |   ✓    |
|      | except  |   ✓    |
|  ✓   |  else   |        |
|  ✓   | finally |   ✓    |

    │   ├── BlockingIOError
